# Mysite

Mysite is an learning tutorial web application built with [Python][0] using the [Django Web Framework][1].

The Site has two learning Resources:

* Python
* Machine Learning


## Technology Stack

- Python 3.6.4
- Django 2.0.2


## Demo

(Deployed this project on [Heroku][3])
